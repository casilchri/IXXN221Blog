Block Frames

Before we could even create our website wireframes we made block frames of our website. From doing our user flow diagram we established that we are going to be five pages we need to create for our website. To divide the workup we each designed a block frame for each concept. So we made three different iterations and we each picked a page we would design and when we moved on to the next concept idea we picked a different page to design so that eventually we all designed three different pages for three different concepts.

The way we decided out final block frames was that we all took turns going through each concept page and the person who created that particular block frame would talk about why they make it that way. We noticed that we all have similar ideas of how we would layout each page so it was pretty easy to pick our final block frame pages.
