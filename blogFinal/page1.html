Why can Git be good for a group of designers?

Git can be useful for designers as it is the most commonly used version control system. It tracks for you the changes a person makes in a file which means it records what has been done and you can go back and see the versions you’ve uploaded if needed to look back at previous work. This is good for a group of designers as it is a platform where the group all have access to the work. The work done would not be lost or tampered with. This makes collaboration easier as it allows changes to a file by multiple people making this good for a group of designers.
